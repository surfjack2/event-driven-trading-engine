# LTB Alpha Selection Model

Alpha generation is driven by liquidity expansion and volatility acceleration.

Primary Alpha Sources

Relative Turnover
Volume expansion relative to moving average.

VWAP Deviation
Price distance from VWAP bands.

Momentum
Short term directional movement.

Volatility Expansion
ATR relative to price.

AlphaRankingWorker

score = turnover * volatility

This favors symbols with:

high liquidity
increasing volatility
active participation
