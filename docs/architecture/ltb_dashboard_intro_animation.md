# LTB Dashboard Intro Animation (Optional Feature)

## 1. 개요

LTB Dashboard 접속 시 사용자 경험 및 브랜딩 요소를 제공하기 위해  
시작 애니메이션(Intro Animation)을 표시할 수 있는 구조를 설계한다.

이 기능은 필수 기능이 아니며  
트레이딩 시스템 완성 이후 선택적으로 구현할 수 있는 **UI 확장 기능**이다.

애니메이션 기능은 관리자 설정을 통해 활성화 / 비활성화할 수 있어야 한다.

---

# 2. 기능 목적

목적

- 대시보드 브랜딩 요소 제공
- 사용자 경험 개선
- 개인 사용자용 재미 요소 제공

이 기능은 트레이딩 엔진 동작과 **완전히 분리된 UI 기능**이어야 한다.

---

# 3. 동작 흐름

대시보드 최초 접속

↓

Intro Image Popup 표시

↓

5초 동안 이미지 축소 애니메이션

↓

이미지 사라짐

↓

이미지 위치에서 코인 / 달러 / 금괴 애니메이션 발생

↓

대시보드 메인 화면 표시

---

# 4. Intro Popup 디자인

## 위치

화면 중앙

## 크기

화면의 약 50%

## 기본 이미지

기본 제공 이미지 예

- Skull Image
- Star Wars Style Opening Logo

## 사용자 이미지

관리자가 사용자 정의 이미지를 업로드할 수 있어야 한다.

지원 포맷

- PNG
- JPG

권장 해상도

1024 x 1024

이미지 저장 경로
