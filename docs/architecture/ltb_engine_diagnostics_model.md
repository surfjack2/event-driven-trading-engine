LTB ENGINE DIAGNOSTICS MODEL
Runtime Metrics Specification
--------------------------------------------------

Purpose

Define the metrics model used by the LTB Validation Monitor.

The diagnostics model aggregates runtime events to compute
engine state metrics, signal pipeline metrics, trading statistics,
and strategy performance indicators.

These metrics enable rapid validation of system health,
trading logic behavior, and strategy performance.

--------------------------------------------------
TRADING STATISTICS
--------------------------------------------------

total_trades

Total number of closed trades


win_rate

winning_trades / total_trades


loss_rate

losing_trades / total_trades


profit_factor

sum(profit_trades) / sum(loss_trades)


average_win

mean profit of winning trades


average_loss

mean loss of losing trades


largest_win

maximum trade profit


largest_loss

maximum trade loss


expectancy

expected profit per trade

formula

expectancy =
(win_rate * avg_win)
-
(loss_rate * avg_loss)


max_drawdown

Maximum cumulative equity drawdown


--------------------------------------------------
STRATEGY PERFORMANCE METRICS
--------------------------------------------------

trades

Number of trades per strategy


pnl

Total profit and loss per strategy


win_rate

Strategy win rate


profit_factor

Strategy profit factor


avg_return

Average return per trade


score

Composite score used by strategy allocation

formula

score =
profit_factor * 0.5
+ win_rate * 0.3
+ avg_return * weight_factor
